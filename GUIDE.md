# Lib Downloader — гайд по установке и использованию

Браузерное расширение для скачивания новелл и манги с сайтов группы Lib.

---

## Установка (Chrome / Edge / Яндекс / Opera / Brave)

1. Открой страницу расширений: **chrome://extensions** (Chrome), **edge://extensions** (Edge),
   **browser://extensions** (Яндекс), **opera://extensions** (Opera).
2. Включи переключатель **«Режим разработчика»** (правый верхний угол).
3. Нажми **«Загрузить распакованное»**.
4. Выбери **корневую папку** `lib-downloader` — ту, где лежит `manifest.json` (не вложенную `src`).
5. Расширение появится в списке — готово.

> **Примечание:** при обновлении файлов расширения нажимай кнопку «Обновить» (🔄) напротив него.

---

## Как открыть панель

Нажми иконку расширения **↓** в панели инструментов браузера.

- **Chrome / Edge** — нативная боковая панель сбоку.
- **Яндекс / Opera / Brave** — встроенная панель выезжает справа прямо на странице
  тайтла (нативная боковая там за флагом, поэтому используется встроенная).
  Предупреждение про `YandexSidePanel` можно игнорировать — всё работает.

Если иконки не видно — нажми значок пазла 🧩 в правом верхнем углу и закрепи **Lib Downloader**.

## Пошаговое использование

### 1. Открой страницу тайтла

Перейди на страницу книги или манги на любом из поддерживаемых сайтов.
Например: `https://ranobelib.me/ru/book/12345--nazvanie-novelly`

Панель автоматически подхватит название и обложку. Если страница не является страницей тайтла — появится подсказка «Откройте страницу тайтла».

### 2. Выбери переводчика

В выпадающем списке **«Переводчик»** выбери нужную ветку перевода (если их несколько).

### 3. Выбери главы

Доступны три способа:

- **Выбрать всё** — чекбокс выделяет все свободные главы сразу.
- **Диапазон** — введи «с» и «по», нажми «Применить».
- **Вручную** — отмечай/снимай галочку у каждой главы.

Главы с замком 🔒 — платные или ранний доступ. Они не выбираются и пропускаются при скачивании.

### 4. Выбери форматы

Отметь один или несколько форматов:

**Для новелл:**
| Формат | Открывается в |
|---|---|
| TXT | Любой текстовый редактор |
| MD | Obsidian, VS Code, Typora |
| EPUB | Calibre, Moon+ Reader, Kindle (2022+) |
| FB2 | FBReader, Bookmate |
| JSON | Для разработчиков / обработки |

**Для манги / манхвы:**
| Формат | Открывается в |
|---|---|
| CBZ | Komga, Kavita, YACReader |
| PDF | Любой PDF-просмотрщик |

Для TXT и MD дополнительно можно включить **«Отдельным файлом на главу»** — тогда вместо одного файла на каждую главу сохраняется отдельный файл (`Глава_001.txt`, `Глава_002.txt` …).

### 5. Нажми «Скачать»

Прогресс отображается полосой и строкой статуса внизу панели.
Файлы раскладываются по формату: **Загрузки / Lib Downloader / [название] / [формат]** —
напр. `…/EPUB/`, `…/CBZ/`, `…/Картинки/Глава_001/`.

## Настройки

Нажми **«Настройки»** в нижней части панели:

| Параметр | По умолчанию | Описание |
|---|---|---|
| Лимит запросов/мин | 240 | Ограничивает скорость загрузки, если сайт throttle-ит запросы. 0 = без лимита. Снизь до 60–120, если ловишь ошибки 429 при больших закачках. |
| Качество JPEG | 90% | Качество сжатия картинок для манги (CBZ/PDF). Меньше = меньше размер файла. |
| Резать длинные страницы | Выкл | Для манхвы (вертикальные страницы): режет на части стандартного размера, удобнее для читалок с постраничной прокруткой. |

## Частые вопросы

**Панель не открывается / пустая**
— Убедись, что открыта именно страница тайтла (URL содержит `digits--slug`), а не главная сайта или страница чтения.

**Ошибка при скачивании отдельных глав**
— Платные главы пропускаются автоматически. Если ошибка на свободной главе — попробуй включить лимит запросов (60/мин) в настройках.

**Скачал, но файл не открывается в читалке**
— Для EPUB используй Calibre или Moon+ Reader. Для FB2 — FBReader. Kindle читает EPUB нативно с 2022 года.

**Хочу скачать только новые главы**
— Отметь нужный диапазон вручную или через поле «с / по».
