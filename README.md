# Lib Downloader

Браузерное расширение для скачивания новелл (и в будущем манги) с сайтов
группы **Lib** — RanobeLib, MangaLib, HentaiLib, SlashLib и др. Открывается
боковой панелью-помощником, сам определяет открытый тайтл, даёт выбрать
переводчика, диапазон глав и форматы экспорта.

## Возможности

- 🧩 Боковая панель (Side Panel) — открывается по клику на иконку
- 🔎 Авто-определение тайтла по открытой вкладке
- 📚 **Новеллы** (RanobeLib) → **EPUB, TXT, FB2, JSON** (TXT — можно отдельным файлом на главу)
- 🖼 **Манга / манхва** (MangaLib, HentaiLib, SlashLib) → **CBZ, PDF, исходные картинки**
- 👥 Выбор ветки переводчика (напр. RQM vs другие)
- ✅ Выбор глав: «выбрать всё», диапазон «с N по M», галочки вручную
- 📁 Сохранение в выбранную папку (File System Access API) или в «Загрузки»
- 🎨 Цветовая тема под открытый сайт (RanobeLib — синяя, MangaLib — оранжевая и т.д.)
- 🌗 Авто тёмная/светлая под систему

> Поддержка Firefox — следующий этап (для Chromium всё готово).

## Установка (Chrome / Edge, режим разработчика)

1. Открой `chrome://extensions` (или `edge://extensions`).
2. Включи **Режим разработчика** (правый верхний угол).
3. Нажми **Загрузить распакованное расширение** и выбери папку
   `lib-downloader` (где лежит `manifest.json`).
4. Закрепи иконку на панели. Клик по иконке открывает боковую панель.

## Как пользоваться

1. Открой страницу тайтла на RanobeLib/MangaLib (страница книги или чтения).
2. Нажми иконку расширения — панель подхватит книгу.
3. Выбери переводчика, отметь главы, выбери форматы.
4. (Опц.) Нажми «Выбрать…» и укажи папку сохранения.
5. Нажми **Скачать**.

## Структура

```
lib-downloader/
  manifest.json            — манифест MV3
  src/
    background.js          — service worker (открытие панели)
    sidepanel.html/.css/.js — боковая панель
    lib/
      sites.js   — определение сайта Lib + Site-Id
      api.js     — клиент API (api.cdnlibs.org)
      parse.js   — ProseMirror/HTML → абзацы текста
      zip.js     — минимальный ZIP-врайтер (для EPUB)
      formats/index.js — генераторы TXT/JSON/FB2/EPUB
      fs.js      — выбор папки и запись файлов
    icons/       — иконки
  _dev/          — тестовые скрипты (не нужны для работы)
```

## Технические заметки

- Реальный API всех Lib-сайтов: `https://api.cdnlibs.org/api`, различие — заголовок `Site-Id`
  (ranobelib=3, mangalib=1, slashlib/yaoilib=2, hentailib=4).
- Контент новелл приходит в формате ProseMirror-JSON, парсер разворачивает его в текст.
- EPUB собирается своим ZIP-врайтером без внешних зависимостей (важно для модерации магазина).
- Если `api.cdnlibs.org` недоступен — включи VPN.
