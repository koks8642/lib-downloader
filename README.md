# Lib Downloader

Браузерное расширение для скачивания **новелл и манги/манхвы** с сайтов группы
**Lib** — RanobeLib, MangaLib, HentaiLib, SlashLib и др. Открывается боковой
панелью-помощником, сам определяет открытый тайтл, даёт выбрать переводчика,
диапазон глав и форматы экспорта, сохраняет в выбранную папку.

Без внешних зависимостей — чистый JavaScript (важно для модерации магазинов).

## Возможности

- 🧩 **Боковая панель** — открывается по клику на иконку, не перекрывает страницу
- 🔎 **Авто-определение тайтла** по открытой вкладке
- 📚 **Новеллы** → **EPUB · FB2 · TXT · MD · JSON** (TXT/MD — можно отдельным файлом на главу)
- 🖼 **Манга / манхва** → **CBZ · PDF · исходные картинки**
- 🔒 **Платные главы** (ранний доступ) помечаются и **исключаются** из скачивания —
  не ломают пакет, видно когда откроются
- 👥 **Выбор переводчика** (ветки) — напр. RQM vs другие команды
- ✅ **Выбор глав**: «выбрать всё», диапазон «с N по M», вручную
- ✂️ **Автокадрирование** длинных страниц манхвы под читалки (настраивается)
- ⏱ **Лимит запросов** + ретраи на 429 — не словить бан при больших закачках
- 📁 **Сохранение в «Загрузки»** — раскладка по подпапкам `Lib Downloader/<название>/`
- 🎨 **Тема под сайт**: RanobeLib — синяя, MangaLib — оранжевая, HentaiLib — красная,
  SlashLib — розовая; авто тёмная/светлая под систему

## Поддерживаемые сайты

| Сайт | Тип | Site-Id |
|------|-----|---------|
| ranobelib.me | новеллы | 3 |
| mangalib.me / .org | манга | 1 |
| hentailib.me / .org | манга | 4 |
| slashlib.me / yaoilib.me | манга | 2 |

## Установка

### Из исходников (Chrome / Edge / Яндекс)
1. Открой `chrome://extensions` (или `edge://extensions`).
2. Включи **Режим разработчика** (правый верхний угол).
3. **Загрузить распакованное расширение** → выбери папку `lib-downloader`
   (где лежит `manifest.json`).
4. Клик по иконке открывает боковую панель.

### Сборка пакета для магазина
```
python pack.py        # создаёт dist/lib-downloader-<версия>.zip
```
Zip заливается в Chrome Web Store / Edge Add-ons как новая версия.

## Как пользоваться

1. Открой страницу тайтла на RanobeLib/MangaLib (книга или чтение).
2. Нажми иконку расширения — панель подхватит книгу.
3. Выбери переводчика, отметь главы, выбери форматы.
4. (Опц.) в «Настройках» — лимит запросов, качество JPEG, автокадр.
5. **Скачать**. Файлы появятся в `Загрузки/Lib Downloader/<название>/`.

## Приватность

Расширение **не собирает и не передаёт** никаких персональных данных. Все запросы
идут только к API самих Lib-сайтов (`api.cdnlibs.org`) и их серверам картинок,
файлы сохраняются локально на твоём устройстве. Нет аналитики, нет трекинга.

## Структура

```
lib-downloader/
  manifest.json              — манифест MV3
  src/
    background.js            — service worker (открытие панели)
    sidepanel.html/.css/.js  — боковая панель
    lib/
      sites.js     — определение сайта Lib + Site-Id + цвет темы
      api.js       — клиент API, лимит запросов, детект платных глав
      parse.js     — ProseMirror/HTML → абзацы текста
      zip.js       — минимальный ZIP-врайтер (EPUB/CBZ)
      img.js       — загрузка картинок + автокадрирование
      settings.js  — настройки (chrome.storage)
      fs.js        — выбор папки и запись файлов
      formats/
        index.js   — TXT/MD/JSON/FB2/EPUB
        manga.js   — CBZ
        pdf.js     — PDF из картинок (DCTDecode)
        mobi.js    — заготовка MOBI (в роадмапе, в пакет не входит)
    icons/
  _dev/                       — тесты/черновики (в пакет не входят)
```

## Технические заметки

- Реальный API всех Lib-сайтов: `https://api.cdnlibs.org/api`, различие — заголовок
  `Site-Id`. Сервер картинок — из `/constants?fields[]=imageServers`.
- Контент новелл — ProseMirror-JSON, парсер разворачивает в текст.
- Платная/ранняя глава: `branch.restricted_view.is_open === false`; её контент API
  отдаёт как HTML — перехватывается и помечается как недоступная.
- EPUB/CBZ — свой ZIP-врайтер, PDF — свой генератор, без внешних библиотек.
- Если `api.cdnlibs.org` недоступен — включи VPN.

## Роадмап

- MOBI (байт-точный, с валидацией в Calibre)
- PDF для новелл (со встроенным кириллическим шрифтом)
- Пауза / возобновление / отмена закачки
- Докачка новых глав в существующий файл
- Поддержка Firefox (`sidebar_action`)
